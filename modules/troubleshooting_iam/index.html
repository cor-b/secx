Troubleshooting IAM

Right people, right access, right resources, right time: management frameworks, subject and user identity controls, secrets, auth protocols, cloud IAM, WiFi auth, access control, conditional access, and logging.

overview

Domain 3 (Security Engineering) OBJ 3.1: troubleshoot common issues with identity and access management components. Topics span management frameworks (proofing, PIM), subject access control, user identity control (biometrics, MFA), secrets management, authentication and authorization (SAML, OpenID, OAuth, federation), cloud IAM access and trust policies, WiFi authentication (802.1X, EAP, SAE), access control (SSO, Kerberos, PAM), conditional access, and logging and monitoring.
