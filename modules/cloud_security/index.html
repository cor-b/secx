Cloud Security

Securing cloud end to end: adoption and control strategies, shared responsibility, connectivity, API and CASB protection, and the build/deploy pipeline through containers and serverless.

overview

Domain 2 (Security Operations) OBJ 2.5: securely implement cloud capabilities in an enterprise environment. Topics span implementation and control strategies, the shared responsibility model, connectivity and integration, cloud security considerations (storage, exposure, leakage, remanence), API security, CASB, infrastructure as code, the CI/CD pipeline, container management, and serverless computing.
