Troubleshooting Network Infrastructure

Finding and fixing what breaks the network: observability, network errors, misconfigurations, IPS/IDS issues, alert analysis, DNS and email security, network issues, and cryptographic and PKI issues.

overview

Domain 3 (Security Engineering) OBJ 3.3: troubleshoot complex network infrastructure security issues. Topics span observability (monitor/understand/diagnose), network errors (switching/routing/VPN), misconfigurations (insecure routing, configuration drift), IPS/IDS issues (placement, rules), alert analysis (false positives/negatives), DNS security (DNSSEC, zone transfers, poisoning, sinkholing), email security (SPF/DKIM/DMARC/S-MIME), network issues (NACL, resource exhaustion, DDoS), cryptographic issues (TLS, cipher mismatch), and PKI issues (CA, certificates, revocation, key management).
