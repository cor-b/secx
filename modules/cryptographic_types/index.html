Cryptographic Types

Symmetric and asymmetric cryptography: the algorithms, the trade-offs, digital signatures, and where public-key crypto is used in practice.

overview

Domain 3 (Security Engineering) OBJ 3.8: apply an appropriate cryptographic use case and/or technique. Topics cover symmetric cryptography (one-time pad, lightweight), symmetric algorithms (stream vs block), symmetric considerations (resources, key management), asymmetric cryptography (key pairs, hybrid, code signing), asymmetric algorithms (RSA/DSA/DH/ECC), digital signatures, and asymmetric use cases.
