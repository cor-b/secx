Security in Systems

Implementing security at every stage of a system's lifecycle — from hardware assurance and requirements through testing, CI/CD, and end-of-life management.

overview

Domain 2, Objective 2.2: given a scenario, implement security in the early and subsequent stages of a system's lifecycle. Topics cover hardware and software assurance, supply chain risk, pre- and post-deployment testing, CI/CD security practices, and end-of-life planning.
