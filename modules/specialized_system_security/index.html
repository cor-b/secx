Specialized System Security

Securing the systems that run the physical world: specialized and OT systems, legacy characteristics, security practices, data management, compliance, and critical/commercial/government challenges.

overview

Domain 3 (Security Engineering) OBJ 3.5: secure specialized and legacy systems against threats. Topics span specialized systems (SoC, embedded, IoT, RF), operational technology (HVAC, ICS, SCADA), the characteristics of legacy systems (obsolete, unsupported, unsecurable, constrained), security practices (segmentation, hardening, monitoring), data management (aggregation, analytics, DAM), compliance (regulatory, environmental, safety), and the challenges facing critical services (utilities, transport, healthcare) and commercial/government sectors (manufacturing, finance, defense).
