Automated Security Operations

Using automation to secure the enterprise: vulnerability scanning, scripting fundamentals, the three languages (Bash, PowerShell, Python), schedulers, and workflow automation.

overview

Domain 3 (Security Engineering) OBJ 3.6: use automation to secure the enterprise. Topics span vulnerability scanning and reporting (Tenable.io, QualysGuard), scripting fundamentals (pseudocode, data structures, control structures), the scripting languages Bash (Unix), PowerShell (Windows), and Python (cross-platform), scheduling with Cron and Scheduled Tasks, and workflow automation that chains multi-step security responses.
