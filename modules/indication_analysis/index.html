Indication Analysis

Analyzing artifacts for incident response: infrastructure and metadata, volatile and stored data, reverse engineering, malware analysis, code stylometry, and the cloud workload protection platform.

overview

Domain 4 (Security Operations) OBJ 4.4: analyze data and artifacts in support of incident response. Topics span infrastructure analysis (JTAG, host, network), metadata analysis (files, media, email headers), volatile and non-volatile storage analysis (order of volatility, forensic imaging), reverse engineering (byte code, binary, disassembly, decompilation), malware analysis (sandboxing, detonation, IoC extraction), code stylometry (variant matching, code similarity, attribution), and the cloud workload protection platform (CWPP).
